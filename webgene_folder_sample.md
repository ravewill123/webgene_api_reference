## Folder Description

folder              |    Description   
--------------------|-------------------------------
/admin              |  後台資料夾
/api                |  應用程式接口
/asset              |  網站資源檔案 , e.g. json , xml 
/video              |  網站影片檔
/audio              |  網站音檔
/svg                |  svg 向量圖檔
/sass               |  sass 未編譯檔案
/partial            |  sass 網站共通元件
/page               |  sass 特殊版面css
/css                |  css 檔案
/font               |  引入字體檔 
/img                |  網站圖片檔
/js                 |  javascript檔案
/vendor             |  第三方程式庫
/template           |  網站樣板
/mobile             |  行動裝置站台
/upload             |  後台上傳資料夾
