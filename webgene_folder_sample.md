#WebGene Folder Sample


##資料夾分類

folder              |    Description   
--------------------|-------------------------------
/asset              |  video,audio,svg等其他網站資源
/img                |  圖片
/css                |  css
/script             |  程式
/script /vendor     |  第三方程式庫
/font               |  字型相關
/template           |  樣板
/mobile             |  行動裝置站台
/api                |  API
/upload             |  上傳資料夾
/admin              |  後台資料夾
